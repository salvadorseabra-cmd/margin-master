# Catalog Naming Recommendations — Premium Italian Products

**Date:** 2026-06-15  
**Principle:** Catalog quality over score. Restaurant-owner browseability in 5 years.

---

## Core principles

1. **Anchor on culinary noun + protected designation** — DOP/IGP and product type outlive invoice wording.
2. **Portuguese browseability, Italian culinary terms where standard** — `Água San Pellegrino`, `Prosciutto cotto`, `Gorgonzola DOP dolce`.
3. **Brand in catalog only when it IS the product** — San Pellegrino, Peroni. Charcuterie/cheese brands → alias.
4. **Strip wheel fractions and cut codes** — `1/2`, `1/8` → purchase layer.
5. **Strip supplier SKU debris** — `HC 4,3`, `Nr. 125`, `PNA`, weight ranges, `Assaporami`.
6. **Strip pack/channel always** — `15ud`, `cx`, `*6`, bulk kg on charcuterie.
7. **Keep beverage serving format** — `75cl`, `33cl`.
8. **Keep culinary variants that change kitchen use** — fior di latte, julienne, massima, com pistacchio, punta d'anca oro, dolce.
9. **Title case brand names** — `San Pellegrino` not `sanpellegrino`.

---

## Attribute matrix

| Attribute | Action | Example |
|-----------|--------|---------|
| DOP / IGP | **KEEP** | Gorgonzola DOP, Mortadella IGP |
| Product grade (culinary) | **KEEP** | punta d'anca oro, massima, dolce, scelto |
| Marketing line name | **REMOVE** | Assaporami, Linea Castello (unless multi-SKU) |
| Charcuterie brand prefix | **REMOVE** | Rovagnati -, Rigamonti -, Arrigoni Formaggi - |
| Wheel fraction | **REMOVE** | 1/2, 1/8 |
| Supplier weight/code | **REMOVE** | HC 4,3, 2,5 Kg |
| Case count | **REMOVE** | x 15ud, CX 75CL*15 |
| Beverage brand | **KEEP** | San Pellegrino, Peroni |
| Pasta shape | **KEEP** | paccheri lisci, mezzi paccheri |
